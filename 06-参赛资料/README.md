## 使用步骤

### 第一步：准备 Demo Zip 包

需要打包以下文件为 zip 上传到社区：

从项目根目录选取：
```
02-功能演示/demo.html              ← 核心交互Demo
01-创意方案/smart-soil-guardian.html  ← 创意方案介绍
03-融合方案/smart-soil-demo.html     ← 融合方案Demo
_shared/                            ← 共享资源（字体、JS库）
04-设计素材/                        ← 设计素材图片
```

打包方式：
```bash
# 把需要的文件复制到一个文件夹，然后打包成 zip
# 注意：HTML中的相对路径要正确
```

### 第二步：准备截图（不少于3张）

需要截取以下类型的图片：

1. **TRAE 对话界面截图** - 展示用 TRAE 开发的过程
   - 可以从 TRAE 里截取生成代码的对话
   - 建议截取功能比较复杂的那段对话

2. **代码/项目结构截图** - 展示项目成果
   - IDE 里的项目结构
   - 或者代码运行效果

3. **Demo 运行效果截图** - 展示最终作品
   - demo.html 的功能页面
   - 建议截无人机飞行控制、土质分析、灾害应急这几个页面

截图要求：
- 清晰可见
- 能展示开发过程和最终效果
- 不少于 3 张

### 第三步：补充 Session ID

需要从 TRAE 中复制准确的 Session ID（不少于3个）。

**如何获取 Session ID：**
在 TRAE 中找到对应的对话，双击对话标题即可复制 Session ID。

**建议选取的会话：**

| 序号 | 会话内容 | 大概日期 | Session ID（待补充） |
|------|---------|---------|---------------------|
| 1 | 创意梳理、模块设计、痛点分析 | 2026-06-18 | 497074a4 （待确认） |
| 2 | 后端开发、Flowable工作流、Spring AI | 2026-06-22 | （请从TRAE中复制） |
| 3 | 前端页面、移动端、公告栏系统 | 2026-06-23 | （请从TRAE中复制） |

> 注意：项目文件中记录的 `376508507502266` 可能是用户ID，
> 请以 TRAE 中双击对话复制出来的 Session ID 为准。

### 第四步：填写报名帖子链接

在 `初赛发帖内容.md` 的第5部分，填入你之前报名审核通过的帖子链接。

### 第五步：发帖

1. 打开 TRAE 官方中文社区
2. 进入大赛报名专区
3. 复制 `初赛发帖内容.md` 的内容
4. 上传 Demo Zip 包和截图
5. 添加标签：`社会服务` `社会公益`
6. 发布
