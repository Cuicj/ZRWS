## 一、项目说明

### 1.1 项目简介

**智壤卫士（ZIRANG SHIELD）** 是一款面向土地资源管理与土壤监测的智能审批与监测系统。

系统集成了工作流引擎、GIS地理信息、AI智能分析、无人机采集等核心技术，为土地资源调查、土壤监测、地质灾害预警提供一站式解决方案。

### 1.2 核心功能

| 功能模块 | 说明 |
|----------|------|
| 🌍 土地资源GIS展示 | Cesium 3D地球，支持地形图、影像图切换，GeoJSON数据叠加 |
| 📋 审批工作流 | Flowable 7.0 工作流引擎，BPMN流程设计器 |
| 🤖 AI智能分析 | Spring AI 驱动，土壤分类、岩层分析、灾害预警 |
| 📰 土地资源公告栏 | AI分析时间线，汇总省市县级土地政策动态 |
| 🛰️ 无人机采集管理 | 大疆/极飞无人机SDK对接，航点任务规划 |
| 🔧 设备管理 | 无人机、RTK、雷达等设备状态监控 |
| 📊 数据可视化 | ECharts图表，钻孔柱状图、成分分析图 |
| 🏗️ BPMN流程设计 | bpmn-js在线设计器，支持草稿保存与部署 |
| 🌡️ 气候变暖监测 | 基于IPCC标准，温度异常、极端天气、变暖趋势多维度监测与风险评估 |
| 🏜️ 沙漠化监测 | 基于UNCCD标准，植被覆盖度、干旱指数、风蚀强度监测与风险预警 |
| 🌱 水土流失监测 | 基于SL 190-2007标准，水力/风力侵蚀分级、USLE模型综合评估 |
| 📚 生态标准库 | 涵盖气候变暖、沙漠化、水土流失、生态安全等104条国际/国家标准数据 |

```
客户生命周期价值 (LTV) = 年订阅费 × 平均续约年限 × 增值服务系数

示例（专业版客户）：
- 年订阅费：8万元
- 平均续约年限：4年
- 增值服务：2万元/年
- LTV = 8×4 + 2×4 = 40万元

客户获取成本 (CAC)：
- 营销成本：约2万元/客户
- 实施成本：约1万元/客户
- CAC ≈ 3万元/客户

投资回报率：LTV/CAC = 40/3 ≈ 13倍
```
