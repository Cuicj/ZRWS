## 4. 架构设计

### 4.1 适配器模式架构

```
┌─────────────────────────────────────────────────────────────┐
│                    业务层 (MissionService)                 │
│                     统一调用接口                           │
└─────────────────────────┬───────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────┐
│              DroneAdapter (接口层)                          │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  connect() / disconnect()                           │   │
│  │  takeoff() / land() / returnToHome()               │   │
│  │  startMission() / pauseMission() / cancelMission() │   │
│  │  getStatus() / getBattery() / getPosition()        │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────┬───────────────────────────────────┘
                          │
         ┌────────────────┼────────────────┐
         ▼                ▼                ▼
┌──────────────┐   ┌──────────────┐   ┌──────────────┐
│ DJIM350Adapter│   │ XAGAdapter   │   │SimulatorAdapter│
│   (大疆)     │   │   (极飞)     │   │   (模拟器)   │
└──────────────┘   └──────────────┘   └──────────────┘
         │                │                │
         ▼                ▼                ▼
┌──────────────┐   ┌──────────────┐   ┌──────────────┐
│ DJI SDK      │   │ XAG SDK      │   │ Mock数据     │
│ (USB/串口)   │   │ (MQTT/HTTP)  │   │              │
└──────────────┘   └──────────────┘   └──────────────┘
```

### 4.2 技术对比

| 特性 | 大疆适配器 | 极飞适配器 |
|------|-----------|-----------|
| SDK | DJI Payload SDK | XAG Developer SDK |
| 通信协议 | USB/串口 | MQTT 3.1.1 / HTTP REST |
| 定位 | RTK GPS | RTK GPS (NTRIP) |
| 实时数据 | WebSocket | WebSocket + MQTT |
| 文件传输 | USB直传 | HTTP文件上传 |

## 5. 核心接口列表

### 5.1 飞行控制接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/v1/drone/{id}/takeoff` | POST | 一键起飞 |
| `/api/v1/drone/{id}/land` | POST | 一键降落 |
| `/api/v1/drone/{id}/return-home` | POST | 一键返航 |
| `/api/v1/drone/{id}/status` | GET | 获取状态 |

### 5.2 任务管理接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/v1/missions` | POST | 创建任务 |
| `/api/v1/missions/{id}/start` | POST | 启动任务 |
| `/api/v1/missions/{id}/pause` | POST | 暂停任务 |
| `/api/v1/missions/{id}/cancel` | POST | 取消任务 |

### 5.3 WebSocket推送

| Topic | 说明 |
|-------|------|
| `/topic/mission.{id}.drone` | 无人机遥测数据 |
| `/topic/mission.{id}.gps` | GPS实时轨迹 |
| `/topic/mission.{id}.progress` | 任务进度 |

## 7. 安全考虑

| 安全维度 | 措施 |
|----------|------|
| 设备认证 | 设备SN + Token绑定 |
| API访问 | JWT Token认证 |
| 数据传输 | TLS 1.2+ 加密 |
| 数据存储 | AES加密存储 |
| 飞行安全 | 禁飞区检查、低电量返航 |
