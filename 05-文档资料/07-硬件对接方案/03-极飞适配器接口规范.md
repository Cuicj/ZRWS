
# 极飞适配器接口规范

## 1. 接口概述

本文档定义极飞无人机适配器（XAGAdapter）的接口规范，遵循策略模式设计，与现有大疆适配器保持一致的接口契约，实现设备无关的统一调用。

## 2. 核心接口定义

### 2.1 DroneAdapter 接口

```java
public interface DroneAdapter {
    
    String connect();
    
    void disconnect();
    
    FlightStatus takeoff(int targetAltitude);
    
    FlightStatus land();
    
    FlightStatus returnToHome();
    
    void setWaypoints(List<Waypoint> waypoints);
    
    MissionExecutionResult startMission(String missionId);
    
    void pauseMission();
    
    void resumeMission();
    
    void cancelMission();
    
    FlightStatus getStatus();
    
    CaptureResult triggerCapture();
    
    int getBattery();
    
    GpsPosition getPosition();
    
    String getModel();
    
    String getSerialNumber();
    
    boolean isConnected();
}
```

### 2.2 XAGAdapter 实现类

## 3. 接口方法详解

### 3.1 连接管理

| 方法名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `connect()` | 建立与极飞无人机的连接 | 无 | `String` - 连接会话ID |
| `disconnect()` | 断开连接 | 无 | `void` |
| `isConnected()` | 检查连接状态 | 无 | `boolean` |
| `getModel()` | 获取设备型号 | 无 | `String` - 如 "XAG-P100" |
| `getSerialNumber()` | 获取设备序列号 | 无 | `String` |

### 3.2 飞行控制

| 方法名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `takeoff(targetAltitude)` | 一键起飞 | `targetAltitude`: int (目标高度m) | `FlightStatus` |
| `land()` | 一键降落 | 无 | `FlightStatus` |
| `returnToHome()` | 一键返航 | 无 | `FlightStatus` |

### 3.3 任务管理

| 方法名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `setWaypoints(waypoints)` | 设置航线航点 | `waypoints`: List\<Waypoint\> | `void` |
| `startMission(missionId)` | 启动任务 | `missionId`: String | `MissionExecutionResult` |
| `pauseMission()` | 暂停任务 | 无 | `void` |
| `resumeMission()` | 恢复任务 | 无 | `void` |
| `cancelMission()` | 取消任务 | 无 | `void` |

### 3.4 状态查询

| 方法名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `getStatus()` | 获取完整飞行状态 | 无 | `FlightStatus` |
| `getBattery()` | 获取电池电量 | 无 | `int` (0-100) |
| `getPosition()` | 获取当前GPS位置 | 无 | `GpsPosition` |

### 3.5 数据采集

| 方法名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `triggerCapture()` | 触发传感器采集 | 无 | `CaptureResult` |

## 4. 数据结构定义

### 4.1 Waypoint（航点）

| 字段名 | 类型 | 含义 | 约束 |
|--------|------|------|------|
| lat | double | 纬度 | -90 ~ 90 |
| lng | double | 经度 | -180 ~ 180 |
| alt | int | 高度（米） | 0 ~ 500 |
| speed | int | 飞行速度（m/s） | 0 ~ 20 |
| action | String | 到达后动作 | capture/sample/hover |
| soilSample | boolean | 是否土壤采样 | true/false |
| area | int | 覆盖面积（亩） | >= 0 |

### 4.2 FlightStatus（飞行状态）

| 字段名 | 类型 | 含义 | 枚举值 |
|--------|------|------|--------|
| droneId | String | 设备ID | - |
| status | String | 状态 | idle/flying/paused/returning/landed/error |
| battery | int | 电量百分比 | 0 ~ 100 |
| altitude | double | 当前高度（米） | >= 0 |
| speed | double | 当前速度（m/s） | >= 0 |
| position | GpsPosition | 当前位置 | - |
| coverage | int | 已覆盖面积（亩） | >= 0 |
| waypointIndex | int | 当前航点索引 | >= 0 |
| totalWaypoints | int | 总航点数 | >= 0 |
| signalStrength | int | 信号强度 | 0 ~ 100 |
| satellites | int | 卫星数量 | >= 0 |
| heading | double | 航向角（度） | 0 ~ 360 |
| pitch | double | 俯仰角（度） | -90 ~ 90 |
| roll | double | 横滚角（度） | -180 ~ 180 |
| yaw | double | 偏航角（度） | 0 ~ 360 |

### 4.3 GpsPosition（GPS位置）

| 字段名 | 类型 | 含义 | 约束 |
|--------|------|------|------|
| lat | double | 纬度 | -90 ~ 90 |
| lng | double | 经度 | -180 ~ 180 |
| accuracy | double | 定位精度（米） | >= 0 |
| altitude | double | 海拔高度（米） | - |

### 4.4 CaptureResult（采集结果）

| 字段名 | 类型 | 含义 |
|--------|------|------|
| timestamp | String | 采集时间 |
| position | GpsPosition | 采集位置 |
| sensorData | SensorData | 传感器数据 |
| imagePath | String | 图片路径 |
| gpsEmbedded | boolean | GPS是否嵌入 |

### 4.5 SensorData（传感器数据）

| 字段名 | 类型 | 含义 |
|--------|------|------|
| multispectral | MultispectralData | 多光谱数据 |
| lidar | LidarData | LiDAR数据 |
| soilProbe | SoilProbeData | 土壤探针数据 |

### 4.6 MultispectralData（多光谱数据）

| 字段名 | 类型 | 含义 |
|--------|------|------|
| channels | String[] | 通道名称数组 |
| reflectance | double[] | 反射率数组 |

### 4.7 MissionExecutionResult（任务执行结果）

| 字段名 | 类型 | 含义 |
|--------|------|------|
| success | boolean | 是否成功 |
| message | String | 结果消息 |
| missionId | String | 任务ID |
| startTime | String | 开始时间 |

## 5. 异常处理

### 5.1 异常类型

| 异常类 | 触发条件 |
|--------|----------|
| DroneNotConnectedException | 设备未连接时执行操作 |
| MissionNotFoundException | 任务不存在 |
| InvalidWaypointException | 航点数据无效 |
| LowBatteryException | 电量不足（< 20%） |
| OutOfRangeException | 超出飞行范围 |
| SDKInitializationException | SDK初始化失败 |
| AuthenticationException | 设备认证失败 |

### 5.2 异常处理示例

```java
public FlightStatus takeoff(int targetAltitude) {
    if (!isConnected()) {
        throw new DroneNotConnectedException("无人机未连接");
    }
    
    if (getBattery() < 20) {
        throw new LowBatteryException("电量不足，当前电量: " + getBattery() + "%");
    }
    
    // 调用SDK起飞
    XAGTakeoffResponse response = sdkClient.takeoff(targetAltitude);
    
    if (!response.isSuccess()) {
        throw new SDKInitializationException(response.getErrorMessage());
    }
    
    currentStatus.setStatus("flying");
    currentStatus.setAltitude(targetAltitude);
    return currentStatus;
}
```

```xml
    <version>1.2.5</version>
</dependency>
```

### 6.2 SDK初始化

```java
@Configuration
public class XAGSDKConfig {
    
    @Value("${zrws-collect.drone.xag.app-key}")
    private String appKey;
    
    @Value("${zrws-collect.drone.xag.app-secret}")
    private String appSecret;
    
    @Value("${zrws-collect.drone.xag.mqtt-broker}")
    private String mqttBroker;
    
    @Value("${zrws-collect.drone.xag.mqtt-port}")
    private int mqttPort;
    
    @Bean
    public XAGSDKClient xagSdkClient() {
        return new XAGSDKClient.Builder()
                .appKey(appKey)
                .appSecret(appSecret)
                .mqttBroker(mqttBroker)
                .mqttPort(mqttPort)
                .build();
    }
    
    @Bean
    public MqttClient mqttClient(XAGSDKClient sdkClient) {
        return sdkClient.getMqttClient();
    }
}
```

## 7. 适配器工厂扩展

### 7.1 工厂类设计

```java
@Component
public class DroneAdapterFactory {
    
    private final Map<String, DroneAdapter> adapters;
    
    private static final Map<String, String> XAG_FAMILY_MAP = Map.of(
        "XAG-P100-Pro", "XAG-P100",
        "XAG-P80", "XAG-P100",
        "XAG-V40", "XAG-V40",
        "XAG-M600", "XAG-M600"
    );
    
    public DroneAdapterFactory(Map<String, DroneAdapter> adapters) {
        this.adapters = adapters;
    }
    
    public DroneAdapter create(String model) {
        // 1. 精确匹配
        if (adapters.containsKey(model)) {
            return adapters.get(model);
        }
        
        // 2. 兼容匹配（型号族）
        String family = XAG_FAMILY_MAP.get(model);
        if (family != null && adapters.containsKey(family)) {
            return adapters.get(family);
        }
        
        // 3. 回退模拟器
        return adapters.get("SIMULATOR");
    }
}
```

## 8. 与大疆适配器对比

| 接口方法 | 大疆适配器 | 极飞适配器 | 差异说明 |
|----------|-----------|-----------|----------|
| connect() | USB/串口连接 | MQTT连接 | 通信方式不同 |
| takeoff() | 同步返回 | 异步返回 | SDK设计差异 |
| triggerCapture() | 本地触发 | 远程指令 | 架构差异 |
| getStatus() | 实时读取 | 订阅推送 | 获取方式不同 |
