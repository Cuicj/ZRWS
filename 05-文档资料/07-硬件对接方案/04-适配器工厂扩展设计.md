
# 适配器工厂扩展设计

## 1. 设计概述

本文档描述无人机适配器工厂（DroneAdapterFactory）的扩展设计，支持极飞无人机的自动适配和动态切换。

## 2. 工厂架构

### 2.1 整体架构

```
┌──────────────────────────────────────────────────────────────┐
│                    DroneAdapterFactory                       │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  设备型号映射表 (DEVICE_FAMILY_MAP)                   │   │
│  │  ┌────────────────────────────────────────────────┐  │   │
│  │  │ DJI-M350 → DJI-M350                          │  │   │
│  │  │ DJI-M300 → DJI-M350 (兼容)                    │  │   │
│  │  │ XAG-P100 → XAG-P100                          │  │   │
│  │  │ XAG-P80  → XAG-P100 (兼容)                   │  │   │
│  │  │ XAG-V40  → XAG-V40                          │  │   │
│  │  │ XAG-M600 → XAG-M600                         │  │   │
│  │  └────────────────────────────────────────────────┘  │   │
│  └──────────────────────────────────────────────────────┘   │
│                            │                               │
│                            ▼                               │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  适配器注册表 (adapters Map<String, DroneAdapter>)   │   │
│  │  ┌─────────────┐ ┌─────────────┐ ┌───────────────┐  │   │
│  │  │ DJI-M350    │ │ XAG-P100    │ │ SIMULATOR     │  │   │

└──────────────────────────────────────────────────────────────┘
```

### 2.2 设计模式

| 模式 | 应用场景 | 说明 |
|------|----------|------|
| 策略模式 | 适配器接口 | 统一接口，多种实现 |
| 工厂模式 | 适配器创建 | 根据型号动态创建 |
| 注册表模式 | 适配器管理 | Spring自动注册 |

## 3. 扩展后的工厂实现

### 3.1 设备型号映射

```java
@Component
public class DroneAdapterFactory {
    
    private final Map<String, DroneAdapter> adapters;
    
    private static final Map<String, String> DJI_FAMILY_MAP = Map.ofEntries(
        Map.entry("DJI-M350-RTK", "DJI-M350"),
        Map.entry("DJI-M300-RTK", "DJI-M350"),
        Map.entry("DJI-M210", "DJI-M350")
    );
    
    private static final Map<String, String> XAG_FAMILY_MAP = Map.ofEntries(
        Map.entry("XAG-P100-Pro", "XAG-P100"),
        Map.entry("XAG-P80", "XAG-P100"),
        Map.entry("XAG-P60", "XAG-P100"),
        Map.entry("XAG-V40-Pro", "XAG-V40"),
        Map.entry("XAG-M600-Pro", "XAG-M600")
    );
    
    private static final Map<String, String> DEVICE_FAMILY_MAP = new HashMap<>();
    
    static {
        DEVICE_FAMILY_MAP.putAll(DJI_FAMILY_MAP);
        DEVICE_FAMILY_MAP.putAll(XAG_FAMILY_MAP);
    }
    
    public DroneAdapterFactory(Map<String, DroneAdapter> adapters) {
        this.adapters = adapters;
    }
    
    public DroneAdapter create(String model) {
        if (model == null || model.isBlank()) {
            throw new IllegalArgumentException("设备型号不能为空");
        }
        
        String normalizedModel = normalizeModel(model);
        
        // 1. 精确匹配
        if (adapters.containsKey(normalizedModel)) {
            return adapters.get(normalizedModel);
        }
        
        // 2. 家族匹配
        String family = DEVICE_FAMILY_MAP.get(normalizedModel);
        if (family != null && adapters.containsKey(family)) {
            return adapters.get(family);
        }
        
        // 3. 前缀匹配
        for (String key : adapters.keySet()) {
            if (normalizedModel.startsWith(key) || key.startsWith(normalizedModel)) {
                return adapters.get(key);
            }
        }
        
        // 4. 回退模拟器
        if (adapters.containsKey("SIMULATOR")) {
            return adapters.get("SIMULATOR");
        }
        
        throw new AdapterNotFoundException("未找到适配的无人机适配器: " + model);
    }
    
    private String normalizeModel(String model) {
        return model.toUpperCase().replaceAll("[\\s-_]+", "-").trim();
    }
    
    public Set<String> getSupportedModels() {
        return adapters.keySet();
    }
    
    public boolean isSupported(String model) {
        String normalized = normalizeModel(model);
        return adapters.containsKey(normalized) || DEVICE_FAMILY_MAP.containsKey(normalized);
    }
}
```

```

│                     Spring容器启动                          │
└─────────────────────────┬───────────────────────────────────┘
                          │
                          ▼

                          ▼

└─────────────────────────────────────────────────────────────┘
```

## 6. 使用示例

### 6.1 创建适配器

```java
@Service
public class DroneService {
    
    private final DroneAdapterFactory factory;
    
    public DroneService(DroneAdapterFactory factory) {
        this.factory = factory;
    }
    
    public void executeMission(String droneModel, String missionId) {
        // 创建适配器
        DroneAdapter adapter = factory.create(droneModel);
        
        // 连接设备
        String sessionId = adapter.connect();
        log.info("设备连接成功: {}", sessionId);
        
        // 执行任务
        MissionExecutionResult result = adapter.startMission(missionId);
        
        if (result.isSuccess()) {
            log.info("任务启动成功: {}", missionId);
        }
    }
}
```

### 6.2 检查设备支持

```java
public void validateDevice(String model) {
    if (!factory.isSupported(model)) {
        throw new UnsupportedDeviceException("不支持的设备型号: " + model);
    }
    
    Set<String> supported = factory.getSupportedModels();
    log.info("支持的设备型号: {}", supported);
}
```

## 8. 异常处理

### 8.1 自定义异常

| 异常类 | 触发条件 | HTTP状态码 |
|--------|----------|------------|
| AdapterNotFoundException | 未找到适配的适配器 | 404 |
| UnsupportedDeviceException | 设备型号不支持 | 400 |
| IllegalArgumentException | 输入参数无效 | 400 |

### 8.2 异常处理示例

```java
@RestControllerAdvice
public class DroneExceptionHandler {
    
    @ExceptionHandler(AdapterNotFoundException.class)
    public ResponseEntity<ErrorResponse> handleAdapterNotFound(AdapterNotFoundException e) {
        ErrorResponse error = ErrorResponse.builder()
            .code(20011)
            .message(e.getMessage())
            .build();
        return ResponseEntity.status(HttpStatus.NOT_FOUND).body(error);
    }
    
    @ExceptionHandler(UnsupportedDeviceException.class)
    public ResponseEntity<ErrorResponse> handleUnsupportedDevice(UnsupportedDeviceException e) {
        ErrorResponse error = ErrorResponse.builder()
            .code(20012)
            .message(e.getMessage())
            .build();
        return ResponseEntity.status(HttpStatus.BAD_REQUEST).body(error);
    }
}
```
