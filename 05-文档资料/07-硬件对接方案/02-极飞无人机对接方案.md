
# 极飞无人机对接方案

## 1. 方案概述

### 1.1 目标
为智壤卫士系统添加极飞（XAG）无人机适配能力，实现极飞农业无人机的飞行控制、数据采集与任务管理功能。

### 1.2 适用设备
| 设备型号 | 系列 | 支持功能 |
|----------|------|----------|
| XAG P100 Pro | 植保无人机 | 飞行控制、多光谱采集、RTK定位 |
| XAG P80 | 植保无人机 | 飞行控制、多光谱采集 |
| XAG V40 | 巡检无人机 | 飞行控制、高清摄像 |
| XAG M600 | 测绘无人机 | 飞行控制、LiDAR扫描 |

### 1.3 技术选型
- **SDK**: 极飞开发者平台 SDK (XAG Developer SDK)
- **通信协议**: MQTT 3.1.1 / HTTP REST API
- **定位**: RTK GPS (支持NTRIP协议)
- **数据传输**: WebSocket (实时遥测) / HTTP (文件上传)

### 2.2 适配器接口设计

| 接口方法 | 功能说明 | 参数 | 返回值 |
|----------|----------|------|--------|
| `connect()` | 连接无人机 | 无 | `boolean` |
| `disconnect()` | 断开连接 | 无 | `void` |
| `takeoff(altitude)` | 一键起飞 | `altitude`: number (目标高度，单位m) | `Promise<void>` |
| `land()` | 一键降落 | 无 | `Promise<void>` |
| `returnToHome()` | 一键返航 | 无 | `Promise<void>` |
| `setWaypoints(points)` | 设置航线航点 | `points`: Waypoint[] | `void` |
| `startMission()` | 执行任务 | 无 | `Promise<void>` |
| `pauseMission()` | 暂停任务 | 无 | `void` |
| `resumeMission()` | 恢复任务 | 无 | `void` |
| `cancelMission()` | 取消任务 | 无 | `void` |
| `getStatus()` | 获取飞行状态 | 无 | `FlightStatus` |
| `triggerCapture()` | 触发传感器采集 | 无 | `CaptureResult` |
| `getBattery()` | 获取电池电量 | 无 | `number` (0-100) |
| `getPosition()` | 获取当前位置 | 无 | `GpsPosition` |

### 2.3 数据结构定义

#### 2.3.1 Waypoint (航点)
```json
{
  "lat": 28.456720,        // 纬度
  "lng": 112.835210,       // 经度
  "alt": 120,              // 高度 (m)
  "speed": 15,             // 飞行速度 (m/s)
  "action": "capture",     // 到达后动作: capture/sample/hover
  "soilSample": true,      // 是否进行土壤采样
  "area": 50               // 覆盖面积 (亩)
}
```

#### 2.3.2 FlightStatus (飞行状态)
```json
{
  "droneId": "UAV-XAG-P100-001",
  "status": "flying",      // idle/flying/paused/returning/landed
  "battery": 85,           // 电量百分比
  "altitude": 120,         // 当前高度 (m)
  "speed": 15,             // 当前速度 (m/s)
  "position": {            // 当前位置
    "lat": 28.456720,
    "lng": 112.835210,
    "accuracy": 0.1        // 定位精度 (m)
  },
  "coverage": 350,         // 已覆盖面积 (亩)
  "waypointIndex": 5,      // 当前执行航点索引
  "totalWaypoints": 10,    // 总航点数
  "signalStrength": 90     // 信号强度 (0-100)
}
```

## 3. API接口规范

### 3.1 飞行控制API

| API路径 | HTTP方法 | 所属Controller | 功能描述 |
|---------|----------|----------------|----------|
| `/api/v1/drone/{droneId}/connect` | POST | FlightController | 连接无人机 |
| `/api/v1/drone/{droneId}/disconnect` | POST | FlightController | 断开连接 |
| `/api/v1/drone/{droneId}/takeoff` | POST | FlightController | 一键起飞 |
| `/api/v1/drone/{droneId}/land` | POST | FlightController | 一键降落 |
| `/api/v1/drone/{droneId}/return-home` | POST | FlightController | 一键返航 |
| `/api/v1/drone/{droneId}/status` | GET | FlightController | 获取飞行状态 |
| `/api/v1/drone/{droneId}/battery` | GET | FlightController | 获取电池电量 |
| `/api/v1/drone/{droneId}/position` | GET | FlightController | 获取当前位置 |

#### 3.1.1 起飞接口示例

**请求**:
```
POST /api/v1/drone/UAV-XAG-P100-001/takeoff
Content-Type: application/json

{
  "targetAltitude": 120,
  "missionId": "m_20240115_001"
}
```

**响应**:
```json
{
  "code": 200,
  "message": "起飞指令已发送",
  "data": {
    "droneId": "UAV-XAG-P100-001",
    "status": "flying",
    "targetAltitude": 120
  },
  "traceId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "timestamp": 1705297800000
}
```

### 3.2 任务管理API

| API路径 | HTTP方法 | 所属Controller | 功能描述 |
|---------|----------|----------------|----------|
| `/api/v1/missions` | POST | MissionController | 创建采集任务 |
| `/api/v1/missions/{missionId}` | GET | MissionController | 查询任务详情 |
| `/api/v1/missions/{missionId}/waypoints` | POST | MissionController | 设置任务航点 |
| `/api/v1/missions/{missionId}/start` | POST | MissionController | 启动任务 |
| `/api/v1/missions/{missionId}/pause` | POST | MissionController | 暂停任务 |
| `/api/v1/missions/{missionId}/resume` | POST | MissionController | 恢复任务 |
| `/api/v1/missions/{missionId}/cancel` | POST | MissionController | 取消任务 |

## 4. MQTT通信协议

### 4.1 连接配置

| 参数 | 值 | 说明 |
|------|-----|------|
| Broker | `mqtt.xag.io` | 极飞MQTT服务器 |
| Port | 1883 / 8883 | 普通端口 / TLS端口 |
| Client ID | `zrws-{droneId}-{timestamp}` | 客户端标识 |
| Username | 设备SN | 极飞设备序列号 |
| Password | 设备Token | 设备访问令牌 |
| Clean Session | true | 清理会话 |
| Keep Alive | 60s | 心跳间隔 |

### 4.2 主题规范

| 主题 | 方向 | 说明 |
|------|------|------|
| `xag/{deviceSn}/command` | 服务端→设备 | 发送控制指令 |
| `xag/{deviceSn}/status` | 设备→服务端 | 设备状态上报 |
| `xag/{deviceSn}/telemetry` | 设备→服务端 | 遥测数据上报 |
| `xag/{deviceSn}/capture` | 设备→服务端 | 采集数据上报 |
| `xag/{deviceSn}/error` | 设备→服务端 | 错误信息上报 |

### 4.3 指令格式

#### 4.3.1 起飞指令
```json
{
  "command": "takeoff",
  "requestId": "req_001",
  "payload": {
    "altitude": 120,
    "speed": 5
  },
  "timestamp": 1705297800000
}
```

#### 4.3.2 设置航点指令
```json
{
  "command": "set_waypoints",
  "requestId": "req_002",
  "payload": {
    "waypoints": [
      {"lat": 28.456720, "lng": 112.835210, "alt": 120},
      {"lat": 28.457000, "lng": 112.835500, "alt": 120}
    ]
  },
  "timestamp": 1705297800000
}
```

## 5. 错误码规范

| 错误码 | 错误信息 | 所属模块 |
|--------|----------|----------|
| 20001 | 无人机未连接 | 飞行控制 |
| 20002 | GPS信号丢失 | 定位 |
| 20003 | 传感器读取失败 | 传感器 |
| 20004 | 任务不存在 | 任务管理 |
| 20005 | 任务状态不允许此操作 | 任务管理 |
| 20006 | 电量不足，无法起飞 | 飞行控制 |
| 20007 | 超出飞行范围 | 飞行控制 |
| 20008 | 设备通信超时 | 通信 |
| 20009 | 极飞SDK初始化失败 | SDK |
| 20010 | 设备认证失败 | 认证 |

## 6. 安全考虑

### 6.1 认证与授权
- 设备连接需使用极飞官方设备Token
- API访问需携带JWT Token
- 设备SN与Token绑定校验

### 6.2 数据加密
- MQTT通信支持TLS 1.2+
- 敏感数据传输加密
- 存储数据AES加密

### 6.3 飞行安全
- 禁飞区检查（起飞前）
- 电量过低自动返航
- 信号丢失自动返航
- 飞行高度限制

## 7. 部署与配置

### 7.1 Nacos配置示例

```yaml
zrws-collect:
  drone:
    default-model: XAG-P100
    xag:
      enabled: true
      mqtt-broker: mqtt.xag.io
      mqtt-port: 8883
      api-url: https://api.xag.io/v1
      app-key: ${XAG_APP_KEY}
      app-secret: ${XAG_APP_SECRET}
    simulator-enabled: false
```

### 7.2 环境变量

| 变量名 | 说明 | 示例值 |
|--------|------|--------|
| XAG_APP_KEY | 极飞开发者平台App Key | xag_abc123 |
| XAG_APP_SECRET | 极飞开发者平台App Secret | secret_xyz789 |
| XAG_MQTT_BROKER | MQTT服务器地址 | mqtt.xag.io |
| XAG_MQTT_PORT | MQTT端口 | 8883 |
