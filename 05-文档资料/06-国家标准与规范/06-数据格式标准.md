# 数据格式与交换标准

## 一、点云数据格式

| 格式 | 扩展名 | 版本 | 组织/厂商 | 说明 | 状态 |
|------|--------|------|-----------|------|------|
| LAS | .las | 1.4 | ASPRS | 公开标准，通用性强 | ✅ 已支持 |

| PCD | .pcd | 0.7 | Point Cloud Library | PCL库原生格式 | ✅ 已规划 |
| XYZ | .xyz | - | 通用 | 简单文本格式(仅坐标) | ✅ 已规划 |
| PTS | .pts | - | 通用 | 带颜色的文本格式 | ⬜ 待实现 |
| E57 | .e57 | - | ASTM | 工业级点云 | ⬜ 待研究 |

| Classification | classification | 分类代码 | uchar | ✅ 已规划 |
| Scan Angle | scan_angle | 扫描角 | short | ✅ 已规划 |

| R, G, B | red, green, blue | 颜色值 | ushort | ⬜ 待实现 |

| 高程基准 | 大地水准面/椭球高 | 必填 | ✅ 已规划 |
| 边界范围 | 四至坐标 | 必填 | ✅ 已规划 |
| 空值 | 无数据值 | 必填 | ✅ 已规划 |
