## 三、面积计算公式

### 3.1 水平投影面积计算
```
S水平 = S实测 × cos(θ)
其中：θ = 地面坡度角
```

### 3.2 坡面面积计算
```
S坡面 = S水平 × 修正系数
修正系数 = 1 / cos(θ)
```

### 3.3 椭球面面积计算（高精度）
```
使用高斯-勒让德积分法计算椭球面投影面积
适用于大面积测量（如省/市级汇总）
```

## 五、大小亩偏差处理

| 参数名称 | 标准值 | 说明 | 状态 |
|----------|--------|------|------|
| 大小亩偏差容忍度 | ±3% | GPS面积与登记面积允许偏差 | ✅ 已预设 |
| 偏差处理方式 | 自动预警+人工核验 | 超限则标记异常 | ✅ 已规划 |

**偏差计算公式：**
```
偏差值 = GPS面积 - 登记面积
偏差率 = 偏差值 / 登记面积 × 100%
```

**偏差处理规则：**

| 偏差率范围 | 处理方式 | 状态 |
|-----------|----------|------|
| ≤3% | 正常，标记为"normal" | ✅ 已预设 |
| 3%-10% | 预警，标记为"review"，需人工核验 | ✅ 已预设 |
| >10% | 异常，标记为"abnormal"，需现场复核 | ✅ 已预设 |
