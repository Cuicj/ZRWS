# 审批与工作流标准

## 一、预设审批流程模板

### 1.1 标准审批流程(STANDARD)

| 步骤序号 | 步骤名称 | 角色 | SLA时限 | 允许操作 | 说明 | 状态 |
|----------|----------|------|---------|----------|------|------|
| 1 | 设计提交 | surveyor | 4h | 提交/撤回 | 外业人员提交采集成果 | ✅ 已预设 |
| 2 | 校核 | checker | 8h | 通过/驳回 | 校核员核查数据质量 | ✅ 已预设 |
| 3 | 审核 | reviewer | 24h | 通过/驳回 | 技术负责人审核 | ✅ 已预设 |
| 4 | 批准 | approver | 48h | 通过/驳回 | 主管领导最终批准 | ✅ 已预设 |
| 5 | 归档 | archiver | - | 确认 | 自动归档或人工归档 | ✅ 已预设 |

## 二、审批角色定义

| 角色代码 | 角色名称 | 权限级别 | 可操作范围 | 继承关系 | 状态 |
|----------|----------|----------|-----------|----------|------|
| field_staff | 外业人员 | 1-基础 | 任务提交、数据采集 | - | ✅ 已预设 |
| surveyor | 测量员 | 2-中级 | 设计提交、成果编制 | field_staff | ✅ 已预设 |
| checker | 校核员 | 3-校核 | 数据校验、质量评估 | surveyor | ✅ 已预设 |
| reviewer | 审核员 | 4-审核 | 审批流转、报告审核 | checker | ✅ 已预设 |
| approver | 审批人 | 5-批准 | 最终批准、驳回 | reviewer | ✅ 已预设 |
| commander | 指挥员 | 4-应急 | 应急审批决策 | reviewer | ✅ 已预设 |
| admin | 系统管理员 | 99-系统 | 全系统管理 | - | ✅ 已预设 |

## 三、审批操作定义

| 操作代码 | 操作名称 | 后续步骤 | 结果状态 | 说明 | 状态 |
|----------|----------|----------|----------|------|------|
| SUBMIT | 提交 | 进入校核 | PROCESSING | 发起审批流程 | ✅ 已预设 |
| APPROVE | 通过 | 进入下一审核 | PASSED | 当前步骤审批通过 | ✅ 已预设 |
| REJECT | 驳回 | 结束流程 | REJECTED | 驳回至申请人 | ✅ 已预设 |
| RETURN | 退回 | 返回上一步 | RETURNED | 退回修改 | ✅ 已预设 |
| WITHDRAW | 撤回 | 结束流程 | WITHDRAWN | 申请人主动撤回 | ✅ 已预设 |

## 四、审批状态流转

```
[起草] → SUBMIT → [审批中]
                        │
        ┌───────────────┼───────────────┐
        ▼               ▼               ▼
     APPROVE         REJECT          RETURN
        │               │               │
        ▼               ▼               ▼
    [校核中]      [已驳回] ←──── [退回]
        │                            │
        ▼                            │
     (后续步骤...)                    │
        │                            │
        └────────────────────────────┘
                       RETURN
```

| 状态代码 | 状态名称 | 说明 | 颜色标识 |
|----------|----------|------|----------|
| DRAFT | 起草 | 还未提交 | #9B59B6 |
| PROCESSING | 审批中 | 正在审批流程中 | #3498DB |
| PASSED | 已通过 | 审批全部通过 | #27AE60 |
| REJECTED | 已驳回 | 被审批人驳回 | #E74C3C |
| RETURNED | 已退回 | 退回修改 | #F39C12 |
| WITHDRAWN | 已撤回 | 申请人主动撤回 | #95A5A6 |
| ARCHIVED | 已归档 | 归档完成 | #2ECC71 |
